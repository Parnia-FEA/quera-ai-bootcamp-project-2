# Quera AI Bootcamp – Project 2
Quera AI Bootcamp - Project 2 (Fall 2025)

# Project Overview

This project consists of two main tasks:

## 1. Accident Detection from Video
Binary classification task to detect whether an accident has occurred in a short video clip.

- Input: Video (sequence of frames)
- Output: Accident / No Accident

---

## 2. Sentiment Analysis on Reviews
Multi-class classification task to predict user sentiment (1 to 5 stars).

- Input: Review text
- Output: Rating (1–5)
